# Zircon_Raphael_ServerWebManage

​	基于Raphael的皓石传奇三，其服务端因为跨平台的原因去掉了使用Dev的winform界面，但也导致了管理不便，故增加了服务端的web管理功能。

​	使用了TabBlazor开发，完成了框架、通讯、系统基础功能，其它模块业务逻辑部分没有做完，有想法的同学可以继续完善。
